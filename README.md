# 🎟️ Event Ticket Booking System

A full-stack event ticket booking system built with:
- **React** (Frontend)
- **Node.js + Express** (Backend)
- **MongoDB** (Database)
- **Razorpay** (Payment Gateway)

## 🧪 Technologies Used

- React + TailwindCSS
- Node.js + Express
- MongoDB
- Razorpay
- CORS
- dotenv for environment variables
